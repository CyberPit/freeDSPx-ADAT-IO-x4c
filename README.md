# freeDSPx-ADAT-IO-x4

BOARD STATUS: WORK IN PROGRESS - DO NOT BUILT!

The freeDSPx ADAT IO x4 is a digital IO expansion board for the freeDSP family. 
It provides 4 Lightpipe inputs and 4 Lightpipe outputs via Toslinks.
